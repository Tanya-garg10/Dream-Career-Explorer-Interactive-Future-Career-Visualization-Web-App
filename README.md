# 🚀 Dream Career Explorer

An interactive web experience where users select their dream career and instantly visualize their future journey. Designed for students and aspirants, it combines storytelling, minimal clicks, and engaging visuals for a smooth and inspiring experience.

![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=flat&logo=javascript&logoColor=black)

## ✨ Features

- **6 Career Paths** — Software Engineer, Doctor, UX/UI Designer, Data Scientist, Entrepreneur, and Research Scientist
- **Interactive Timeline** — Each career unfolds as a staggered-reveal journey with milestones, from foundation years to leadership
- **Inspirational Quotes** — Every career path opens with a motivational quote from an industry icon
- **Skill Tags** — See the key skills you'll build along the way
- **Particle Background** — Subtle animated canvas with floating dots and faint connections
- **Smooth Animations** — Screen transitions, timeline reveals, and skill tag pop-ins
- **Fully Responsive** — Works seamlessly on mobile, tablet, and desktop
- **Keyboard Accessible** — All interactive elements support keyboard navigation
- **Reduced Motion Support** — Respects `prefers-reduced-motion` for accessibility

## 🖥️ Demo

Open `index.html` in any modern browser — no build step or server required.

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Tanya-garg10/Dream-Career-Explorer-Interactive-Future-Career-Visualization-Web-App.git
   cd Dream-Career-Explorer-Interactive-Future-Career-Visualization-Web-App
   ```

2. **Open in browser**
   ```bash
   # Simply open the file
   open index.html

   # Or use a local server
   npx serve .
   ```

No dependencies, no build tools — just open and explore.

## 🎯 How It Works

1. **Welcome Screen** — A hero section invites you to explore careers
2. **Career Selection** — Pick from 6 beautifully designed career cards
3. **Journey Visualization** — Watch your chosen career path unfold as an animated timeline with milestones and skills

## 🛠️ Tech Stack

- **HTML5** — Semantic markup with ARIA attributes
- **CSS3** — Custom properties, grid layout, keyframe animations
- **Vanilla JavaScript** — Zero dependencies, canvas particle system
- **Google Fonts** — Inter + Playfair Display

## 📱 Responsive Design

The layout adapts gracefully across screen sizes:
- **Mobile** — Single-column card grid, compact timeline
- **Tablet** — Two-column grid with balanced spacing
- **Desktop** — Three-column grid with full visual effects
